# Calculator Project

A simple web-based calculator project using **HTML, CSS, and JavaScript**.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (\*)
- Division (/)
- Responsive and user-friendly design
- Works in any modern browser

## How to Run

1. Clone the repository:
   git clone https://github.com/rakibul-web3-dev/calculator.git
2. Open `index.html` in your browser

- Double-click the file
- Or right-click → Open With → Browser
